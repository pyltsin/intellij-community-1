Reports comparisons in which the compareTo method can be replaced to isAfter, isBefore or isEqual of LocalTime, LocalDateTime, LocalDate, OffsetTime or OffsetDateTime.

Example:


  LocalDate date1 = LocalDate.now();
  LocalDate date2 = LocalDate.now();
  boolean t = date1.compareTo(date2) > 0;

After the quick-fix is applied:


  LocalDate date1 = LocalDate.now();
  LocalDate date2 = LocalDate.now();
  boolean t = date1.isAfter(date2);
